# ATM-Machine-Assignment-2-C-
A simple program where a user user is prompted to either create a new account, or login.

# Create Account
User will be prompted to enter a user id and password. The user id must be an int value.

# Login
User will be prompted to enter their user id and password that they entered beforehand.

# User Menu
You can now choose to deposit, withdraw, or request the balance of your account. Values entered for deposit and withdraw must be a double.

# Quit
When choosing to quit you will be brought back to the main menu, where you can login or signup with another account.
